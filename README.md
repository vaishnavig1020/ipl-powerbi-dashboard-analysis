🏏 IPL Data Analysis Dashboard (2008–2025)

An interactive dashboard built using Microsoft Power BI to analyze historical data from the Indian Premier League (IPL).

This project transforms raw IPL match data into clear, interactive visualizations and actionable insights, enabling users to explore team performance, player statistics, and season trends across multiple years.

📊 Dashboard Highlights
🏏 Team Performance Analysis

Compare IPL teams based on total wins, matches played, and seasonal performance trends.

📈 Player Statistics

Analyze top players based on runs scored, wickets taken, and overall match impact.

📅 Season-wise Insights

Explore match outcomes and team performances across IPL seasons from 2008 to 2025.

🎯 Interactive Filters

Use slicers to analyze data dynamically by season, team, and player.

🔗 External Links Integration

Provides quick access to official IPL resources and social media links directly from the dashboard.
